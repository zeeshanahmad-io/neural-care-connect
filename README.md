## üì¶ Getting Started

To set up the project locally, follow these steps:

### Prerequisites

*   Node.js (v18 or higher)
*   npm or Yarn

### Installation

```bash
git clone [YOUR_REPOSITORY_URL_HERE]
cd neural-care-connect
npm install # or yarn install
```

### Running Locally

```bash
npm run dev # or yarn dev
```
